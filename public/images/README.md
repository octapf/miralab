# Portfolio Project Images

Place your portfolio project images in this directory.

Recommended image specifications:
- Format: JPG or PNG
- Resolution: 1200x800px (landscape)
- Size: < 500KB per image
- Names: project1.jpg, project2.jpg, etc.

Current placeholders in portfolio data:
- project1.jpg
- project2.jpg
- project3.jpg
- project4.jpg
- project5.jpg
- project6.jpg

You can use any image URL or replace these with your actual project screenshots.
